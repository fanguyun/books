# ch03 大类资产配置框架

这一部分对应第二部分前半，包括“改造美林投资时钟”“美元破百，冲击中国”“世界大宗商品周期研究”。

## 核心结论

- 传统投资时钟过于简化，必须加入长波、汇率和跨资产传导。
- 美元、商品与全球风险偏好之间存在重要联动。

## 关键框架

### 1. 改造美林投资时钟
- 不只看增长和通胀，还要看康波位置、房地产、汇率和库存周期。

### 2. 美元是重要外生变量
- 美元强弱会通过资本流动、风险偏好和商品价格影响中国与全球资产。

### 3. 商品周期
- 商品更容易体现全球工业化、信用环境和通胀预期的共振。

## 使用提醒

- 当你做资产配置时，先问自己：是不是只在套模板，没有看长波背景。
