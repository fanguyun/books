# 术语表

## 位 / bit
- 数字系统中的最小信息单位，在本书语境里通常表示为 0 或 1。

## 字节 / byte
- 8 个位组成的常见基本存储单位。

## 编码 / decoding
- 把现实数据映射成数字表示称为编码；把数字表示解释回有意义内容称为解码。

## ASCII / Unicode
- 文本字符的标准数字表示方式；Unicode 比 ASCII 覆盖更多语言和字符。

## 模拟 / 数字
- 模拟值可连续变化；数字值用有限状态集合表示。

## 逻辑门
- 实现 AND、OR、NOT 等逻辑运算的数字电路基本单元。

## 晶体管
- 数字电路的基础开关器件，可用来构造逻辑门和存储单元。

## 锁存器 / 触发器
- 能记住状态的一位存储电路，是更大存储系统的基础。

## RAM
- 随机存取存储器，CPU 可直接访问，用于运行时保存程序和数据。

## CPU
- 中央处理器，负责解释并执行机器指令。

## 寄存器
- CPU 内部的小型高速存储位置，用于保存操作数、中间结果和状态。

## 机器码
- CPU 能直接执行的二进制指令。

## 汇编语言
- 机器指令的人类可读表示，每条语句通常对应一条机器指令。

## 编译 / 链接
- 编译把高级语言转换为目标代码；链接把目标文件组合成可执行文件。

## 操作系统
- 管理进程、内存、文件和设备等资源的系统软件层。

## TCP/IP
- 互联网通信的基础协议套件。

## HTTP
- Web 上传输资源的应用层协议。
