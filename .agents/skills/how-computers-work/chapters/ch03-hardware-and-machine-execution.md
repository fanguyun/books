# ch03 硬件与机器执行

这一部分对应第 7、8 章，讲 RAM、CPU、寄存器、机器码和汇编。

## 核心结论

- 计算机之所以通用，是因为硬件能执行程序，而不是只执行硬连线功能。
- CPU 读取内存中的机器指令并操作寄存器、主存和 I/O。

## 关键框架

### 1. RAM / CPU / I/O 三角
- 主存保存运行数据，CPU 执行指令，I/O 连接外界。

### 2. 指令层级
- 机器码是最终可执行形式，汇编是其可读映射。

### 3. 地址与寄存器
- 内存地址让 CPU 能定位数据，寄存器提供高速临时存储。

## 使用提醒

- 解释性能、程序执行或底层 bug 时，要沿着“指令从哪来、数据去哪、CPU 在做什么”来讲。
