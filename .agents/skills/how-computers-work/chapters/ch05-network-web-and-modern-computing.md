# ch05 网络、Web 与现代计算

这一部分对应第 11 至 13 章，讲互联网、万维网、app、虚拟化和云计算。

## 核心结论

- 互联网和 Web 不是一回事，前者是网络基础设施，后者是其中一种资源系统。
- 现代 app、云计算和虚拟化，都是在已有计算抽象层上进一步叠加的产物。

## 关键框架

### 1. TCP/IP 分层
- 链路层、网络层、传输层、应用层各司其职。

### 2. Web 基本链路
- URL 定位资源，浏览器发 HTTP 请求，服务器响应内容。

### 3. 现代形态
- 本机 app、Web app、PWA、虚拟化和云，都是对分发、运行和隔离方式的不同选择。

## 使用提醒

- 解释“打开网页发生了什么”时，不要把 DNS、TCP、HTTP、HTML 渲染混成一步。
