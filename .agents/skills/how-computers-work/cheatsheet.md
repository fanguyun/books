# 速查表

## 解释计算机基础时先问

- 这个问题卡在数据表示、硬件、电路、系统还是网络？
- 需要先向下解释一层，还是先向上建立全局图？
- 对方缺的是定义、流程还是层次感？

## 解释 0 和 1 时

- 它被编码成什么位模式？
- 这些位由什么介质保存？
- 这个表示是文本、数字还是图像/音频？

## 解释程序执行时

- 这是源代码、汇编还是机器码？
- 谁把它转换成 CPU 能执行的形式？
- CPU 需要从哪里取指令和数据？

## 解释系统与资源时

- 这个行为需要 CPU、RAM、磁盘还是 I/O？
- 是操作系统在管理，还是应用自己在处理？
- 这是临时状态还是持久状态？

## 解释网络与 Web 时

- 这是互联网问题还是 Web 问题？
- 用了什么协议？
- 是客户端发请求，还是服务器处理响应？

## 学习路线提醒

- 不要跳过二进制与数据表示直接学系统。
- 不要把浏览器、互联网和操作系统混成一个概念。
- 遇到复杂名词时，先映射到已有层次框架。
