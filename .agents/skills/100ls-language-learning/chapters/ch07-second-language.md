# ch07 第二外语

## 核心主张

- 100LS 不只适用于英语，也适用于日语、法语、阿拉伯语、泰语等第二外语。
- 差别主要在于是否先需要补齐字母、假名或发音系统。

## 关键内容

- 学日语前，先把平假名和片假名练到能直接认读和书写，再进入日剧 100LS。
- 法语中因为口语和字幕可能不完全一致，动画片等字幕更稳定的素材反而更适合作为训练入口。
- 阿拉伯语、泰语这类“字母看上去很陌生”的语言，先花小块时间征服字母系统，再上影片和音频。
- 第一门语言练成后，后续多语种学习的底层方法高度相似。

## 使用提示

- 遇到陌生字母别急着下结论说难，先分离出“字母系统”和“口语训练”两个问题。
- 第二外语初期先选你最喜欢的剧、动画、短视频或老师音频。
- 当你已经能梦到自己在说那门语言时，通常说明输入量开始足够了。
