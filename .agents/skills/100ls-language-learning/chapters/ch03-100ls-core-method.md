# ch03 100LS 核心方法

## 核心主张

- 外语要像母语一样，通过情境、重复和身体参与来学。
- 100LS 的本质是：用真实素材做高频听说循环，直到句子在耳朵和嘴巴里变成反射。

## 五步流程

1. 无字幕完整看一遍，先接触真实语流。
2. 中文字幕确认意思和剧情。
3. 目标语言字幕锁定不会的片段并查清。
4. 分片段反复听和说，模仿速度、连读、语气。
5. 练到不看画面也能复现，再进入下一段。

## 关键原则

- 外语不是靠头脑分析，而是靠训练获得。
- 片段比整片重要，复现比理解重要。
- 电影和剧集之所以好用，是因为它们天然提供情境和真实对白。

## 使用提示

- 初期不要频繁换片；一部能坚持做完比十部随便看过更有效。
- 听不懂时不要急着怀疑自己，先暂停、拆段、重放。
- 只要持续练下去，某个时点会出现“突然听清楚”的突破感。
