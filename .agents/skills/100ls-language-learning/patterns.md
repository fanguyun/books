# Patterns

## 1. 100 天英语启动模式

适用：长期学了很多年但听不懂、说不出的人。

步骤：
1. 先写下“我学英语是为了什么机会”。
2. 选一部你愿意反复看的日常对话型电影或剧集。
3. 按 `无字幕 -> 中文字幕 -> 英文字幕 -> 片段重复` 的顺序推进。
4. 每天固定做一段高强度听说训练，通勤时间只听对应音频。
5. 第一轮完成后，再上新闻或 TED。

## 2. 开不了口修复模式

适用：单词和语法学过很多，但见到真人开不了口。

步骤：
1. 暂停继续刷新语法点。
2. 选短对话密集、生活化的素材。
3. 把目标从“看懂”改成“跟上并说出来”。
4. 大声练、录音练、照镜子练。
5. 只要一句跟不上，就拆更小片段继续练。

## 3. 瓶颈期续航模式

适用：练到十几遍或几十遍后烦躁、怀疑、想换方法。

步骤：
1. 先承认停滞期是正常阶段。
2. 缩短单次训练片段，降低启动阻力。
3. 保持每天不断线，不追求当日明显进步。
4. 用“今天只搞定一个片段”替代“我要完成整部片”。
5. 把进度判断标准改成是否更能跟上、是否更少卡顿。

## 4. 第二外语迁移模式

适用：已完成第一门外语训练，准备迁移到日语、法语或其他语言。

步骤：
1. 先解决字母、假名或发音系统。
2. 选最有兴趣的剧集、电影或动画片。
3. 使用同样的 100LS 拆片段方法。
4. 遇到字形陌生，不先下结论说自己不行。
5. 完成第一轮素材后，再扩大到更多内容和文化输入。

## 5. TED 进阶模式

适用：已经能靠电影素材训练，想提升思想密度和正式表达。

步骤：
1. 选一支 18 到 20 分钟、主题真正感兴趣的演讲。
2. 先完整看，再切成 10 秒片段反复练。
3. 重点模仿演讲者的节奏、停顿、逻辑连接和关键词表达。
4. 练到能不看字幕跟上为止。
5. 顺手吸收演讲中的观点，兼做表达素材库。
