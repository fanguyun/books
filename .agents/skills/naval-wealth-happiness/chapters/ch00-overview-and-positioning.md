# ch00 总览与定位

## 本章回答什么问题

- 《纳瓦尔宝典》到底想解决什么？
- 财富、金钱、地位为什么要拆开看？
- 为什么纳瓦尔把“把自己产品化”当成总纲？

## 核心结论

### 1. 先区分财富、金钱和地位
- 财富是持续生产价值的资产。
- 金钱只是价值交换媒介。
- 地位是社会排名游戏。
- 一旦把三者混在一起，人就容易用高收入或高头衔假装自己在接近财富自由。

### 2. 财富创造是一种技能
- 纳瓦尔反复强调，赚钱不是纯靠辛苦或运气，而是可以学习的技能。
- 这门技能的重点不是“更卖力”，而是“知道做什么、和谁做、何时做，以及如何规模化”。

### 3. 总纲是“把自己产品化”
- “自己”指独特性、专长、人格信誉和责任感。
- “产品化”指让价值可复制、可规模化，而不是只靠一次性交付。
- 这句话本质上把整本书的财富主线压缩成了一个检验问题：你是否正在把独特价值转化成能被放大的系统。

### 4. 长期主义不是附加项，而是默认前提
- 合作、技能、声誉、习惯和资本回报都遵循复利。
- 因此，短期套利式思维很难和纳瓦尔框架兼容。

## 使用提示

- 讨论职业、项目、创业时，先回到这一章，避免一开始就落入收入或地位讨论。
- 如果一个方案不能被解释成“独特价值 + 规模化机制”，大概率还没到位。
