# 速查表

## 冲突前先问

| 问题 | 如果答案偏左 | 更接近的动作 |
|---|---|---|
| 我想要的是长期成长，还是此刻服从？ | 此刻服从 | 先停，重想目标 |
| 我是在理解需要，还是只在管行为？ | 只管行为 | 先问原因 |
| 我是在保护关系，还是在赢输对抗？ | 赢输对抗 | 降温、修复连接 |
| 我说“不”是必要边界，还是出于便利/面子/焦虑？ | 便利/面子/焦虑 | 重新考虑要求 |

## 避免的句式

| 旧句式 | 问题 | 替代方向 |
|---|---|---|
| “你再这样我就……” | 威胁 | 说真实边界与理由 |
| “你已经选择了被罚。” | 伪选择 | 诚实承担成人决定 |
| “你怎么这么坏/懒/不懂事。” | 人格化批评 | 聚焦具体行为 |
| “快说对不起。” | 强迫道歉 | 先理解影响，再谈修复 |
| “考了多少分？” | 结果导向 | 问理解、兴趣、困难 |

## 必须设限时

1. 说明真实理由。
2. 语气尽量温和。
3. 给能给的选择。
4. 不把孩子逼进羞耻与报复。
5. 事后补回尊严和自主感。

## 当孩子失控时

1. 先保证安全。
2. 不急着讲理。
3. 不把升级当权威挑战。
4. 等情绪过去，再讨论诱因和修复。

## 当你自己快失控时

1. 承认自己在赶时间、焦虑或怕失控。
2. 先不做惩罚决定。
3. 能改环境就先改环境。
4. 如果已经做过头，后续修复要明确。
