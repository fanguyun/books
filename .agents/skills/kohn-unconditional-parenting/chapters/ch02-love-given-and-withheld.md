# Chapter 2: 爱·给予和撤回

## Core Idea
许多看似温和的做法，本质上仍是在让孩子感受“爱会被拿走”，包括暂停、冷处理、表扬和奖励。

## Frameworks Introduced
- **爱的撤回**
  - 何时用：分析冷落、失望脸、暂停、忽视、减少亲近时。
  - 如何看：孩子感受到的不是“教育”，而是“关系被收回”。
- **奖励失败**
  - 何时用：当成人想用奖品、表扬、积分塑造行为时。
  - 如何看：奖励不增强责任感，只增强对外部回报的依赖。

## Key Concepts
- **暂停 / 隔离**：常被包装为理性冷静期，实则常是情感驱逐。
- **正面强化**：看似鼓励，实则把认可与表现挂钩。
- **自我评价条件化**：孩子慢慢相信“我做对时才值得被喜欢”。

## Mental Models
- 孩子不只会从惩罚里学东西，也会从“被夸时”和“被冷时”学东西。
- 表扬如果带有操控意味，并不比批评更自由。
- 问题不是家长有没有好意，而是孩子如何经历这种好意。

## Anti-patterns
- **把暂停当中性工具**：忽略其关系伤害和羞耻效应。
- **频繁表扬**：让孩子做事越来越为了成人反应，而非出于价值认同。
- **让认可随表现波动**：制造脆弱自尊与讨好倾向。

## Key Takeaways
1. 爱的撤回比体罚更隐蔽，但破坏力不低。
2. 奖励和表扬会把注意力从“事情本身”挪到“成人怎么反应”。
3. 真正稳的关系，是冲突后仍能让孩子感到被接纳。
