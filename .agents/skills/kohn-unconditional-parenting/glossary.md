# 术语表

**爱的撤回** — 用冷落、暂停、隔离、失望表情或减少亲近感来让孩子感受“失去爱”的控制手段。（Ch 2, Ch 8）

**并肩协作** — 不把冲突当制服孩子，而把它当共同理解问题、修复关系、寻找方案的过程。（Ch 1, Ch 7）

**补偿性自主感支持** — 当某件事必须由成人拍板时，在其他可控维度补回孩子的选择权与尊严。（Ch 9）

**长期目标** — 家长真正想培养出的人的样子，如有道德感、独立思考、责任感、共情能力。（Ch 7）

**伪选择** — 看似给孩子选择，实则用威胁、误导或预设后果迫使其按成人意志行动。（Ch 9）

**分数导向** — 把学习目标从理解、探索、兴趣转成可量化结果与外部评价。（Ch 5）

**关系优先** — 在处理行为前，先保护亲子信任与安全感，把关系视为教育的前提。（Ch 7）

**行为主义** — 只盯外显行为，并用强化与惩罚解释和操控人的行动的心理学取向。（Ch 1）

**控制型养育** — 以顺从、效率和可管理性为优先，用权力压过理解和协商。（Ch 3, Ch 6）

**无条件养育** — 孩子不因行为、情绪、成绩或顺从程度失去被爱与被接纳的资格。（Ch 1, Ch 8）

**无条件认可** — 孩子确信自己作为一个人被珍视，而不是只在“表现正确”时被珍视。（Ch 1, Ch 8）

**外在动机** — 为获得奖励、避免惩罚、赢得评价而行动，而不是因为认同价值或内在兴趣。（Ch 2, Ch 5）

**真实选择** — 孩子在对结果有实际影响的事情上拥有发言权，而非装饰性决定。（Ch 9）

**暂停 / 隔离** — 常被包装为温和纪律，实质上仍可能被孩子体验为爱的撤回与惩罚。（Ch 2, Ch 4）

**正面强化** — 用表扬、奖品、积分等方式塑造行为，看似积极，实则常是更柔软的控制。（Ch 2）

**证据不足时假定孩子无辜** — 在未确认恶意前，优先按年龄、能力不足、信息不全或无伤大雅的解释理解行为。（Ch 7, Ch 10）

**自我评价条件化** — 孩子逐渐相信自己只有在成功、听话、讨人喜欢时才有价值。（Ch 2, Ch 5）

**实施手段** — 把养育理解成对孩子施加奖励、惩罚、威胁和操控，以换取顺从。（Ch 1, Ch 4）

**视角转换** — 透过孩子的感受、理解水平和现实限制去重新定义冲突，而不是只从成人便利出发。（Ch 10）

**限制批评** — 控制批评的次数、范围和强度，避免把具体问题演化为人格否定。（Ch 8）
