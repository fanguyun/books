# Chapter 2: 互惠

## Core Idea
当别人先给了我们某种好处，我们会感到心理上的回报义务，这会显著提高顺从率。

## Frameworks Introduced
- **先给后要**
  - 何时用：需要提高合作、成交、回复或接受请求时。
  - 如何看：对方先让你拿到好处，再提出要求。
- **拒绝-退让**
  - 何时用：谈判、募捐、议价。
  - 如何看：先提大请求，被拒后退到较小请求，让你也更难拒绝。

## Key Concepts
- **义务感**：互惠的心理核心。
- **让步交换**：对方先退一步，你也更想配合。
- **不请自来的恩惠**：即使你没主动要，收到后仍可能觉得欠了人情。

## Anti-patterns
- **把赠品看成纯好意**：忽略其后续回报压力。
- **只看礼物大小**：很多时候关键不是价值，而是“先给了你”。

## Key Takeaways
1. 互惠会制造一种难以忽视的回报冲动。
2. 让步本身也是一种“给予”。
3. 防御方法不是拒绝所有好意，而是识别“好意是否被设计成交换”。
