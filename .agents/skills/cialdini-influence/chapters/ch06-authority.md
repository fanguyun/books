# Chapter 6: 权威

## Core Idea
人对权威的服从常常是自动的，而且不仅真正专家有效，头衔、制服和身份标志这些权威符号也能显著提高顺从。

## Frameworks Introduced
- **真正专家 vs 权威符号**
  - 何时用：判断某个“专家建议”是否值得跟从。
  - 如何看：要区分真实资格与外观包装。
- **两步防御**
  - 何时用：碰到权威压力时。
  - 如何看：先问“他是真专家吗”，再问“他说的是真话吗”。

## Key Concepts
- **头衔**：最常见也最容易伪造的权威符号。
- **制服**：能快速触发服从程序。
- **身份标志**：豪车、讲究穿着、专业外观都能增强顺从。

## Anti-patterns
- **见到专业外观就默认其专业意见相关**。
- **把权威感误认成真实性和善意**。

## Key Takeaways
1. 权威影响力常被人严重低估。
2. 很多顺从不是服从知识，而是服从符号。
3. 最稳的防御是检查资格相关性与利益关系。
