# Chapter 3: 承诺和一致

## Core Idea
一旦人公开、主动地作出承诺，就会倾向于保持前后一致，从而让后续顺从变得更容易。

## Frameworks Introduced
- **登门槛效应**
  - 何时用：希望对方逐步接受更大请求时。
  - 如何看：先让对方答应一个小请求，再扩大。
- **低球策略**
  - 何时用：先吸引承诺，再改变条件。
  - 如何看：真正生效的是已作出的决定和自我形象，而不只是最初条件。

## Key Concepts
- **一致性压力**：人想被视为稳定、理性、有原则。
- **公开承诺**：公开性会显著提高后续一致行动。
- **自我说服**：最强的承诺通常不是别人灌输，而是自己写下、说出或选择。

## Anti-patterns
- **为了看起来一致而死扛错误决定**。
- **把小承诺当小事**：它往往是更大转化链条的起点。

## Key Takeaways
1. 承诺越主动、越公开、越书面化，影响越强。
2. 一致性常让人继续一条原本已不划算的路径。
3. 防御时要问：我是在忠于价值，还是只是在维护面子上的一致？
