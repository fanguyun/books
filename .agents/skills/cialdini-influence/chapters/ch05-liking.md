# Chapter 5: 喜好

## Core Idea
我们更容易答应自己喜欢的人，而“喜欢”常由相似、赞美、熟悉和关联等因素快速建立。

## Frameworks Introduced
- **相似性吸引**
  - 何时用：建立信任、降低陌生感。
  - 如何看：共同背景、经历、偏好会提高认同。
- **赞美降低防御**
  - 何时用：需要让对方更开放时。
  - 如何看：即使知道对方在恭维，赞美也常仍然有效。

## Key Concepts
- **熟悉效应**：越熟悉越容易喜欢。
- **关联效应**：积极对象会把好感转移给相关人或物。
- **朋友式销售**：先做人情，再做交易。

## Anti-patterns
- **把对人的好感误判成对方案的认可**。
- **硬凑关系和相似性**：会适得其反。

## Key Takeaways
1. 喜好原则作用于“人”，不直接作用于“事实”。
2. 相似、赞美和关联是最常见的喜好构建器。
3. 防御时要分开判断“我喜欢这个人”和“这件事值得答应”。
