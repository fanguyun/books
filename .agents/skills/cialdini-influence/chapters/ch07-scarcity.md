# Chapter 7: 稀缺

## Core Idea
东西一旦显得更少、更快失去或更难获得，人就会更想要它；稀缺原则特别善于制造紧迫决策。

## Frameworks Introduced
- **失去厌恶**
  - 何时用：解释稀缺为何比单纯“更好”更有力。
  - 如何看：人对失去机会的痛感常强于获得收益的快乐。
- **机会窗口**
  - 何时用：分析限时、限量、最后名额等说法。
  - 如何看：重点不只是稀少，而是“现在不做就没了”。

## Key Concepts
- **数量稀缺**：库存、名额、配额有限。
- **时间稀缺**：仅今天、最后一天、即将截止。
- **竞争性稀缺**：别人也在抢，会进一步推高价值感。

## Anti-patterns
- **看到限时就自动提高价值判断**。
- **把“怕错过”当成“真的需要”**。

## Key Takeaways
1. 稀缺会让思考模式从评估价值变成抢救机会。
2. 真正危险的不是稀缺本身，而是它压缩独立判断的时间。
3. 防御时要把注意力从“快没了”拉回“值不值得”。
