# Chapter 4: 社会认同

## Core Idea
在不确定情境中，人会通过观察别人来决定什么是正确的，因此“别人也这么做”本身就是强信号。

## Frameworks Introduced
- **社会认同**
  - 何时用：陌生产品、高不确定场景、缺少个人判断标准时。
  - 如何看：人数、相似性、一致性共同决定说服力。
- **相似他人最有效**
  - 何时用：面向特定人群的传播与转化。
  - 如何看：和目标对象越像的案例，影响越大。

## Key Concepts
- **从众线索**：销量、评价、案例、排队、热榜。
- **多元无知**：每个人都在模仿别人，结果群体共同走错。
- **旁观者效应**：越多人在场，越可能没人行动。

## Anti-patterns
- **拿数量代替质量判断**。
- **在错误群体样本上做从众判断**。

## Key Takeaways
1. 不确定性越高，社会认同越强。
2. 看到“很多人都这么做”时，先问他们是不是“对我有参考意义的人”。
3. 社会认同既能提高转化，也能制造群体性盲点。
