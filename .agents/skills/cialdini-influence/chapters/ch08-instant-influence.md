# 尾声：即时的影响力

## Core Idea
理解影响力原则的真正价值，不只是会用，还在于能在被触发的当下识别它们，并恢复自主判断。

## Frameworks Introduced
- **即时识别**
  - 何时用：在真实交易、说服、营销和社交场景里。
  - 如何看：一旦意识到自己被某条原则激活，就先暂停。
- **从触发器退回真实问题**
  - 何时用：觉得自己被推进节奏时。
  - 如何看：回到价值、需求、证据和代价本身。

## Key Concepts
- **警觉性**：知道自己会被哪些原则影响。
- **重获判断**：把决策从自动顺从拉回主动评估。
- **影响伦理**：会用原则不代表应该无边界地用。

## Key Takeaways
1. 六大原则最危险的时候，往往是它们无声无息地协同出现。
2. 防御不是拒绝所有影响，而是识别不必要的自动顺从。
3. 最高效的策略常常也是最需要伦理边界的策略。
