# 引言

## Core Idea
全书的研究问题是：为什么相同请求换一种说法，就会显著提高“说是”的概率。

## Frameworks Introduced
- **顺从研究**
  - 何时用：分析别人如何让人答应请求时。
  - 如何看：不是只看内容本身，还要看请求的表达方式与触发线索。
- **参与式观察**
  - 何时用：理解本书证据来源时。
  - 如何看：作者不仅做实验，还潜入销售、筹款、广告等行业内部观察。

## Key Concepts
- **顺从心理学**：研究人为何会答应请求。
- **专业顺从者**：销售员、筹款者、广告商等靠影响别人吃饭的人。
- **影响力武器**：能稳定提高顺从概率的心理原则。

## Key Takeaways
1. 本书不只研究“说服”，更研究“顺从”。
2. 职业说服者比普通人更懂得如何触发自动反应。
3. 研究这些原则，既是为了使用，也是为了防御。
