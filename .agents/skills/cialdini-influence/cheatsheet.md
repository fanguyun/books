# 速查表

## 快速识别

| 对方在做什么 | 可能触发的原则 | 先问自己什么 |
|---|---|---|
| 先送赠品、样品、优惠 | 互惠 | 我是不是因此更难拒绝了？ |
| 让我先答应一件小事 | 承诺和一致 | 这是不是在为更大请求铺路？ |
| 强调“大家都在买/都这么做” | 社会认同 | 这些“别人”是不是和我相关？ |
| 先建立亲近感、赞美我 | 喜好 | 我是在答应这个人，还是答应这件事？ |
| 搬出专家、机构、头衔 | 权威 | 这是真正相关的专家吗？ |
| 强调限时、限量、最后机会 | 稀缺 | 这是真稀缺，还是在催我别思考？ |

## 两个高频防御问题

1. 我现在是不是进入了自动模式？
2. 如果把这些触发器拿掉，我还会做同样决定吗？

## 权威防御

- 这个人是真专家吗？
- 他的专业资格和当前问题真的相关吗？
- 他是否会因我的顺从直接受益？

## 稀缺防御

- 我是真的想要，还是怕失去？
- 如果明天还能买，我今天还会买吗？
- 时间压力是不是被故意设计出来的？

## 一句话提醒

- 互惠会制造义务。
- 一致性会制造自我约束。
- 社会认同会制造从众安全感。
- 喜好会降低戒心。
- 权威会压缩独立判断。
- 稀缺会放大损失敏感。
